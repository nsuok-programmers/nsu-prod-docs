    STVMAJR

    Major, Minor, Concentration Validation Table

    Definition Table

    This is a validation/lookup table that defines valid codes for other tables.

    Schema

    Column Type Description
    STVMAJR_CODE VARCHAR2(4 CHAR) This field identifies the major code referenced in the Catalog, Class Sched., Recruit., Admissions, Gen. Student, Registr., and Acad. Hist. Modules. Reqd. value: 00 - Major Not Declared.
    STVMAJR_DESC VARCHAR2(30 CHAR) This field specifies the major area of study associated with the major code.
    STVMAJR_CIPC_CODE VARCHAR2(6 CHAR) This field identifies the Classification of Institutional Programs (CIP) code associated with the major area of study.
    STVMAJR_VALID_MAJOR_IND VARCHAR2(1 CHAR) This field indicates whether the area of study is a valid major.
    STVMAJR_VALID_MINOR_IND VARCHAR2(1 CHAR) This field indicates whether the area of study is a valid minor.
    STVMAJR_VALID_CONCENTRATN_IND VARCHAR2(1 CHAR) This field indicates whether the area of study is a valid concentration.
    STVMAJR_ACTIVITY_DATE DATE This field identifies the most recent date a record was created or updated.
    STVMAJR_OCCUPATION_IND VARCHAR2(1 CHAR) Occupation specific indicator. Y indicates program has been designated as occupation specific
    STVMAJR_AID_ELIGIBILITY_IND VARCHAR2(1 CHAR) A indicator which indicates the eligibility for aid
    STVMAJR_SYSTEM_REQ_IND VARCHAR2(1 CHAR) System Required Indicator
    STVMAJR_VR_MSG_NO NUMBER(6,0) The Voice Response message number assigned to the recorded message that describes the major code.
    STVMAJR_DISP_WEB_IND RAW(1 BYTE) Display on Web Indicator is now obsolete.
    STVMAJR_SEVIS_EQUIV VARCHAR2(6 CHAR) SEVIS EQUIVALENCY: SEVIS code for primary major
    STVMAJR_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    STVMAJR_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    STVMAJR_USER_ID VARCHAR2(30 CHAR) USER ID: The user ID of the person who inserted or last updated this record.
    STVMAJR_DATA_ORIGIN VARCHAR2(30 CHAR) DATA ORIGIN: Source system that created or updated the data.
    STVMAJR_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
    STVMAJR_MAJOR_DESC VARCHAR2(100 BYTE) MAJOR DESC: The MAJOR description.
    STVMAJR_MINOR_DESC VARCHAR2(100 BYTE) MINOR DESC: The MINOR description.
    STVMAJR_CONCENTRATION_DESC VARCHAR2(100 BYTE) CONCENTRATION DESC: The CONCENTRATION description.
    STVMAJR_TOPS_CODE VARCHAR2(6 BYTE) Taxonomy of Program Code: Validated by STVTOPS_CODE field and define with same format, VARCHAR2(6).
