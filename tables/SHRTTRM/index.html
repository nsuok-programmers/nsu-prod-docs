    SHRTTRM

    Institutional Course Maintenance Term Header Repeating Table

    Schema

    Column Type Description Definition
    SHRTTRM_PIDM NUMBER(8,0) Internal Identification Number of Student.
    SHRTTRM_TERM_CODE VARCHAR2(6 CHAR) Term Code. Term of student institutional course record. STVTERM
    SHRTTRM_UPDATE_SOURCE_IND VARCHAR2(1 CHAR) Source Indicator. Indicates sourse of how record was created. Valid values are: S - System created during grade roll process, and U - User created.
    SHRTTRM_PRE_CATALOG_IND VARCHAR2(1 CHAR) This field is the Pre-Catalog Indicator. It specifies if the term course information is based on the existing Banner catalog or is conversion data being entered prior to the Banner catalog data. Valid values are: Y - Pre-catalog, and N - Current Catalog.
    SHRTTRM_RECORD_STATUS_IND VARCHAR2(1 CHAR) Status Indicator. Specifies the status of the term record. Valid values are: G - Grade Mailer Pending, C - Grade Mailer processing complete, R - Revised Grade Mailer Pending, N - New Record.
    SHRTTRM_RECORD_STATUS_DATE DATE Status Date. Date associated with the status of the record.
    SHRTTRM_GRADE_MAILING_DATE DATE Grade Mailer Date. Indicates the date grade mailer was sent to the student.
    SHRTTRM_GRADE_MAILER_CHG_DATE DATE Grade Mailer Change Date. Indicates the date a revised or duplicate grade mailer was sent to the student.
    SHRTTRM_GRADE_MAILER_DUP VARCHAR2(1 CHAR) Grade Mailer Duplicate Indicator. Specifies that duplicate grade mailer should be sent to the student. Valid value is Y - Send duplicate mailer.
    SHRTTRM_GRADE_MAILER_DUP_DATE DATE Duplicate Grade Mailer Date. Indicates date duplicate grade mailer was sent.
    SHRTTRM_EXAM_CODE VARCHAR2(2 CHAR) Examination Code. Identifies a special examination the student took during the term.
    SHRTTRM_CODE_TRANSCRIPT_DIST VARCHAR2(2 CHAR) This field is not currently used.
    SHRTTRM_ASTD_CODE_END_OF_TERM VARCHAR2(2 CHAR) This field is the End of Term Academic Standing Code. It Identifies student academic standing for the term (probation, good standing, etc.). Calculated based on the academic standing rules during the end of term standing process or may be entered directly. STVASTD
    SHRTTRM_ASTD_DATE_END_OF_TERM DATE Date Academic Standing was entered. STVASTD
    SHRTTRM_ACTIVITY_DATE DATE Activity Date. Specifies the date record was created or updated.
    SHRTTRM_ASTD_CODE_DL VARCHAR2(2 CHAR) Deans List Code. Identifies if student qualified for deans list for the term. Calculated based on academic standing deans list rules or may be directly entered. STVASTD
    SHRTTRM_ASTD_DATE_DL DATE Date Deans List Status was entered. STVASTD
    SHRTTRM_WRSN_CODE VARCHAR2(2 CHAR) Reason for withdrawal from this institution. STVWRSN
    SHRTTRM_SBGI_CODE_TRANS VARCHAR2(20 CHAR) Institution to which the student will transfer STVSBGI
    SHRTTRM_PREV_CODE VARCHAR2(2 CHAR) Academic History Term record Progress Evaluation code. This code is populated by SHRPREV.
    SHRTTRM_PREV_DATE DATE Academic History Term record Progress Evaluation date. This is the date progress evaluation code was updated.
    SHRTTRM_CAST_CODE VARCHAR2(2 CHAR) Academic History Term record Combined Academic Standing code. This code is populated by SHRPREV.
    SHRTTRM_CAST_DATE DATE Academic History Term record Combined Academic Standing date. This is the date combined academic standing code was updated.
    SHRTTRM_SCPS_CODE VARCHAR2(6 CHAR) Student Centric Period code.
    SHRTTRM_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    SHRTTRM_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    SHRTTRM_USER_ID VARCHAR2(30 CHAR) USER ID: The user ID of the person who inserted or last updated this record.
    SHRTTRM_DATA_ORIGIN VARCHAR2(30 CHAR) DATA ORIGIN: Source system that created or updated the data.
    SHRTTRM_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
    SHRTTRM_GUID VARCHAR2(36 BYTE) GUID: Global Unique Identifier to uniquely identify the record of a resource for integration.
