    SORLCUR

    Student Curriculum repeating table to store level, college, program and other data associated with curriculum.

    Schema

    Column Type Description Definition
    SORLCUR_PIDM NUMBER(8,0) PIDM: Unique internal ID for individual who has records in the system.
    SORLCUR_SEQNO NUMBER(4,0) LCUR SEQNO: One up number to define the row.
    SORLCUR_LMOD_CODE VARCHAR2(15 CHAR) LEARNER MODULE CODE: Learner Module Code. STVLMOD
    SORLCUR_TERM_CODE VARCHAR2(6 CHAR) TERM CODE: Term code. STVTERM
    SORLCUR_KEY_SEQNO NUMBER(2,0) KEY SEQNO: Sequence number of key record, saradap_appl_no, srbrecr_seqno, shrdgmr_seqno, sgrstsp_key_seqno.
    SORLCUR_PRIORITY_NO NUMBER(4,0) PRIORITY NO: priority of the curriculum within the module.
    SORLCUR_ROLL_IND VARCHAR2(1 CHAR) ROLL IND: Y/N indicator if the learner curriclum should roll to academic history when courses are rolled.
    SORLCUR_CACT_CODE VARCHAR2(15 CHAR) CURRICULUM ACTIVITY STATUS: Activity status code for the curriculum. STVCACT
    SORLCUR_USER_ID VARCHAR2(30 CHAR) UPDATE USER ID: Most recent user to create or update the record.
    SORLCUR_DATA_ORIGIN VARCHAR2(30 CHAR) DATA SOURCE: Source system that generated the data
    SORLCUR_ACTIVITY_DATE DATE ACTIVITY DATE: The most recent date a record was created or updated
    SORLCUR_LEVL_CODE VARCHAR2(2 CHAR) LEVEL CODE: Level code of the curriculum. STVLEVL
    SORLCUR_COLL_CODE VARCHAR2(2 CHAR) COLLEGE CODE: college code of the curriculum. STVCOLL
    SORLCUR_DEGC_CODE VARCHAR2(6 CHAR) DEGREE CODE: degree code of the curriculum. STVDEGC
    SORLCUR_TERM_CODE_CTLG VARCHAR2(6 CHAR) TERM CATALOG: catalog term code of the learner curriculum. STVTERM
    SORLCUR_TERM_CODE_END VARCHAR2(6 CHAR) TERM END: end term code of the curriculum. STVTERM
    SORLCUR_TERM_CODE_MATRIC VARCHAR2(6 CHAR) TERM MATRICULATED: term code learner matriculated into the curriculum. STVTERM
    SORLCUR_TERM_CODE_ADMIT VARCHAR2(6 CHAR) TERM ADMIT: term code learner was admitted to curriculum. STVTERM
    SORLCUR_ADMT_CODE VARCHAR2(2 CHAR) ADMIT CODE: admit code for the learners admission to the curriculum. STVADMT
    SORLCUR_CAMP_CODE VARCHAR2(3 CHAR) CAMPUS CODE: campus code of the curriculum. STVCAMP
    SORLCUR_PROGRAM VARCHAR2(12 CHAR) PROGRAM: program of the curriculum.
    SORLCUR_START_DATE DATE START DATE: date curriculum starts.
    SORLCUR_END_DATE DATE END DATE: date curriculum ends.
    SORLCUR_CURR_RULE NUMBER(8,0) CURRICULUM RULE NUMBER: rule number of the curriculum.
    SORLCUR_ROLLED_SEQNO NUMBER(4,0) ROLLED SEQNO: sorlcur_seqno of outcome created during the grade roll.
    SORLCUR_STYP_CODE VARCHAR2(1 CHAR) LEARNER TYPE: This describes the type of student. STVSTYP
    SORLCUR_RATE_CODE VARCHAR2(5 CHAR) RATE CODE: This is the fee assessment rate code. STVRATE
    SORLCUR_LEAV_CODE VARCHAR2(1 CHAR) LEAVE CODE: This describes the reason for the leave of absense. STVLEAV
    SORLCUR_LEAV_FROM_DATE DATE LEAVE TO DATE: This is the end date of the leave.
    SORLCUR_LEAV_TO_DATE DATE LEAVE FROM DATE: This is the start date of the leave.
    SORLCUR_EXP_GRAD_DATE DATE EXPECTED GRADUATION DATE: This is the date the learner is expected to graduate from the curriculum.
    SORLCUR_TERM_CODE_GRAD VARCHAR2(6 CHAR) EXPECTED GRADUATION TERM: This is the expected term the learner will graduate. STVTERM
    SORLCUR_ACYR_CODE VARCHAR2(4 CHAR) GRADUATION ACADEMIC YEAR: This is the academic year for the graduation. STVACYR
    SORLCUR_SITE_CODE VARCHAR2(3 CHAR) LEARNER SITE: This describes the site of student. STVSITE
    SORLCUR_APPL_SEQNO NUMBER(4,0) APPLICATION SEQUENCE NUMBER: This is the sequence number of the application curriculum the learner curriculum was created from.
    SORLCUR_APPL_KEY_SEQNO NUMBER(2,0) APPLICATION NUMBER: This is the application number the learner curriculum was created from.
    SORLCUR_USER_ID_UPDATE VARCHAR2(30 CHAR)
    SORLCUR_ACTIVITY_DATE_UPDATE DATE UPDATE ACTIVITY DATE: Most recent date the record was updated.
    SORLCUR_GAPP_SEQNO NUMBER(4,0) GRADUATION SEQUENCE NUMBER: The sequence number of the graduation application.
    SORLCUR_CURRENT_CDE VARCHAR2(1 CHAR) CURRENT INDICATOR: Will be Y if this is the current curriculum, NULL if not.
    SORLCUR_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    SORLCUR_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    SORLCUR_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
    SORLCUR_GUID VARCHAR2(36 BYTE) Learner Curriculum repeating Table Global Unique Identifier: Stores the global unique identifier for a student program object.
    SORLCUR_OUTCOME_GUID VARCHAR2(36 BYTE) OUTCOME GUID: Global Unique Identifier to uniquely identify the record of a resource for integration.
