    SHRLGPA

    Level GPA Table

    Schema

    Column Type Description Definition
    SHRLGPA_GPA_TYPE_IND VARCHAR2(1 CHAR) GPA Type Indicator. Valid values are: I - Institutional Course GPA, T - Transfer Course GPA, and O - Overall Course GPA.
    SHRLGPA_LEVL_CODE VARCHAR2(2 CHAR) Level Code associated with GPA hours and calculations. STVLEVL
    SHRLGPA_PIDM NUMBER(8,0) Internal Identification Number of Student.
    SHRLGPA_HOURS_ATTEMPTED NUMBER(11,3) Hours Attempted for the level.
    SHRLGPA_HOURS_EARNED NUMBER(11,3) Hours Earned for the level.
    SHRLGPA_GPA_HOURS NUMBER(11,3) Hours Used for GPA calculation for the level.
    SHRLGPA_QUALITY_POINTS NUMBER(17,6) Quality Points for the level. Calculated by multiplying credit hours of the course by quality points associated with the grade.
    SHRLGPA_GPA NUMBER(23,9) GPA for the level. Calculated by dividing GPA hours for the level into Quality points for the level.
    SHRLGPA_ACTIVITY_DATE DATE Activtity Date. Specifies the date record was created or updated.
    SHRLGPA_HOURS_PASSED NUMBER(11,3) Hours Passed for the level.
    SHRLGPA_GPA_CALC VARCHAR2(1 CHAR) This field is used to determine whether the gpa for a student needs to be recalculated or not. Y means the gpa is up to date and N means the student has had courses rolled to history and the gpa needs to be recalulated for them
    SHRLGPA_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    SHRLGPA_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    SHRLGPA_USER_ID VARCHAR2(30 CHAR) USER ID: The user ID of the person who inserted or last updated this record.
    SHRLGPA_DATA_ORIGIN VARCHAR2(30 CHAR) DATA ORIGIN: Source system that created or updated the data.
    SHRLGPA_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
