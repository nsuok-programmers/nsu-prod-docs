    STVASTD

    Academic Standing Validation Table

    Definition Table

    This is a validation/lookup table that defines valid codes for other tables.

    Schema

    Column Type Description
    STVASTD_CODE VARCHAR2(2 CHAR) This field identifies the academic standing code referenced in the General Student, Registration and Academic History Modules. Required Value: 00 - Good Standing.
    STVASTD_DESC VARCHAR2(30 CHAR) This field specifies the academic standing (e.g. deans list, academic warning, probation) associated with the standing code.
    STVASTD_PROBATION_IND VARCHAR2(1 CHAR) This field identifies whether academic standing indicates academic probation. Y - Probation indicated.
    STVASTD_PREVENT_REG_IND VARCHAR2(1 CHAR) This field identifies whether academic standing prohibits registration. Y - Registration prohibited.
    STVASTD_MAX_REG_HOURS NUMBER(9,3) This field identifies the maximum number of hours for which a student may register under the associated academic standing.
    STVASTD_ACTIVITY_DATE DATE This field identifies the most recent date a record was created or updated.
    STVASTD_DEANS_LIST_IND VARCHAR2(1 CHAR) This field identifies whether academic standing indicates deans list.
    STVASTD_SYSTEM_REQ_IND VARCHAR2(1 CHAR) System Required Indicator
    STVASTD_EDI_EQUIV VARCHAR2(3 CHAR) EDI/SPEEDE value for the academic standing code that equates to the acdemic standing code. Refer to the SPEEDE Implementation Guide, with data element #641 for valid values.
    STVASTD_VR_MSG_NO NUMBER(6,0) The Voice Response message number assigned to the recorded message that describes the academic standing code.
    STVASTD_MIN_REG_HOURS NUMBER(9,3) MINIMUM HOURS: This field identifies the minimum number of hours for which a student may register under the associated academic standing.
    STVASTD_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    STVASTD_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    STVASTD_USER_ID VARCHAR2(30 CHAR) USER ID: The user ID of the person who inserted or last updated this record.
    STVASTD_DATA_ORIGIN VARCHAR2(30 CHAR) DATA ORIGIN: Source system that created or updated the data.
    STVASTD_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
    STVASTD_GUID VARCHAR2(36 BYTE) GUID: GUID storage for academic standings resource.

    Students Who are Inelligable to Register

    This will grab any student who is no longer able to register for classes.

    cant_register.sql
    -- Students Who Cannot Register
-- written by Nathan Beene 4/14/26
SELECT
    sgbstdn_pidm,
    spriden_first_name,
    spriden_last_name,
    CASE
        WHEN stvastd_prevent_reg_ind = 'Y' THEN 'N'
        ELSE 'Y'
    END as can_register
FROM
    sgbstdn
LEFT JOIN
    spriden ON spriden_pidm = sgbstdn_pidm
LEFT JOIN
    stvastd ON sgbstdn_astd_code = stvastd_code
WHERE
    spriden_change_ind IS NULL
    AND stvastd_prevent_reg_ind = 'Y'
ORDER BY
    spriden_last_name, spriden_first_name;
