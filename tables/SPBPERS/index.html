    SPBPERS

    Individual Personal Information Table

    Schema

    Column Type Description Definition
    SPBPERS_PIDM NUMBER(8,0) Internal Identification Number of Person.
    SPBPERS_SSN VARCHAR2(15 CHAR) Person's social security number.
    SPBPERS_BIRTH_DATE DATE This field maintains person birth date.
    SPBPERS_LGCY_CODE VARCHAR2(1 CHAR) This field maintains legacy code associated with person. STVLGCY
    SPBPERS_ETHN_CODE VARCHAR2(2 CHAR) ETHNIC CODE: This field identifies the ethnic code referenced on the General Person Form (SPAPERS) and by the Source/Base Institution Year Form (SOABGIY). The code is mapped to other codes which are required for government reporting. STVETHN
    SPBPERS_MRTL_CODE VARCHAR2(1 CHAR) MARITAL CODE: Marital status of person. STVMRTL
    SPBPERS_RELG_CODE VARCHAR2(2 CHAR) NOT USED: This field maintains religious affiliation associated with person. STVRELG
    SPBPERS_SEX VARCHAR2(1 CHAR) This field maintains the sex of person. Valid values are: M - Male, F - Female, N - Unknown.
    SPBPERS_CONFID_IND VARCHAR2(1 CHAR) This field identifies if a person record is confidential Valid value is: Y - confidential.
    SPBPERS_DEAD_IND VARCHAR2(1 CHAR) This field indicates if a person is deceased. Valid value is: Y - deceased.
    SPBPERS_VETC_FILE_NUMBER VARCHAR2(10 CHAR) This field maintains veteran identification number associated with person.
    SPBPERS_LEGAL_NAME VARCHAR2(500 CHAR) This field maintains legal name associated with person.
    SPBPERS_PREF_FIRST_NAME VARCHAR2(60 CHAR) This field maintains the preferred first name associated with person.
    SPBPERS_NAME_PREFIX VARCHAR2(20 CHAR) This field maintains the prefix (Mr, Mrs, etc) used before person name.
    SPBPERS_NAME_SUFFIX VARCHAR2(20 CHAR) This field maintains the suffix (Jr, Sr, etc) used after person name.
    SPBPERS_ACTIVITY_DATE DATE This field defines the most current date a record is added or changed.
    SPBPERS_VERA_IND VARCHAR2(1 CHAR) Veteran Category. (NULL) Not a Veteran, (B) Protected veteran choosing not to self-identify the classification, (O) Active Wartime or Campaign Badge Veteran, (V) Not a Protected Veteran
    SPBPERS_CITZ_IND VARCHAR2(1 CHAR) Citizen Indicator.
    SPBPERS_DEAD_DATE DATE Person Deceased Date.
    SPBPERS_PIN RAW(1 BYTE) Person Identification Number. This column is obsolete, now gobtpac_pin
    SPBPERS_CITZ_CODE VARCHAR2(2 CHAR) Person Citizen Type
    SPBPERS_HAIR_CODE VARCHAR2(2 CHAR) The hair color of the person being defined.
    SPBPERS_EYES_CODE VARCHAR2(2 CHAR) The eye color of the person being defined.
    SPBPERS_CITY_BIRTH VARCHAR2(50 CHAR) The city where the person was born.
    SPBPERS_STAT_CODE_BIRTH VARCHAR2(3 CHAR) The state in which the person was born.
    SPBPERS_DRIVER_LICENSE VARCHAR2(20 CHAR) The Driver License Number as it appears on the actual license.
    SPBPERS_STAT_CODE_DRIVER VARCHAR2(3 CHAR) The State code describing the state the driver license was issued.
    SPBPERS_NATN_CODE_DRIVER VARCHAR2(5 CHAR) The Nation code associated with the Nation the Driver license was issued.
    SPBPERS_UOMS_CODE_HEIGHT VARCHAR2(4 CHAR) The Unit of Measure describing the height value.
    SPBPERS_HEIGHT NUMBER(2,0) The number value describing the height of the person.
    SPBPERS_UOMS_CODE_WEIGHT VARCHAR2(4 CHAR) The Unit of Measure describing the weight value.
    SPBPERS_WEIGHT NUMBER(4,0) The number value describing the weight of the person.
    SPBPERS_SDVET_IND VARCHAR2(1 CHAR) Indicator to identify an individual as a special disabled veteran.
    SPBPERS_LICENSE_ISSUED_DATE DATE The issue date of the individuals driver license.
    SPBPERS_LICENSE_EXPIRES_DATE DATE The expiration date of the individuals driver license.
    SPBPERS_INCAR_IND VARCHAR2(1 CHAR) The indication of the individuals incarceration.
    SPBPERS_WEBID RAW(1 BYTE) Web identification session token is now obsolete.
    SPBPERS_WEB_LAST_ACCESS RAW(1 BYTE) Web last access time stamp is now obsolete.
    SPBPERS_PIN_DISABLED_IND RAW(1 BYTE) Indicates whether or not the PIN number is disabled. This column is obsolete, now gobtpac_pin_disabled_ind
    SPBPERS_ITIN NUMBER(9,0) The international tax id number.
    SPBPERS_ACTIVE_DUTY_SEPR_DATE DATE Active Duty Separation Date: The Date that the person was separated from active duty.
    SPBPERS_DATA_ORIGIN VARCHAR2(30 CHAR) DATA ORIGIN: Source system that created or updated the row
    SPBPERS_USER_ID VARCHAR2(30 CHAR) USER ID: User who inserted or last update the data
    SPBPERS_ETHN_CDE VARCHAR2(1 CHAR) ETHNIC CODE: This field identifies the ethnic code defined by the U.S. government. The valid values are 1 - Not Hispanic or Latino, 2 - Hispanic or Latino, or null. STVETHN
    SPBPERS_CONFIRMED_RE_CDE VARCHAR2(1 CHAR) RACE AND ETHNICITY CONFIRMED: This field identifies the race and ethnicity has been confirmed. Valid values are (Y)es, (N)o and Null.
    SPBPERS_CONFIRMED_RE_DATE DATE RACE AND ETHNICITY CONFIRMED DATE: This field identifies when the race and ethnicity has been confirmed.
    SPBPERS_ARMED_SERV_MED_VET_IND VARCHAR2(1 CHAR) ARMED FORCES SERVICE MEDAL INDICATOR: Armed Forces Service Medal Indicator. Valid values are (Y)es, (N)o. Default value is N.
    SPBPERS_SURROGATE_ID NUMBER(19,0) SURROGATE ID: Immutable unique key
    SPBPERS_VERSION NUMBER(19,0) VERSION: Optimistic lock token.
    SPBPERS_VPDI_CODE VARCHAR2(6 CHAR) VPDI CODE: Multi-entity processing code.
    SPBPERS_GNDR_CODE VARCHAR2(4 CHAR) GNDR CODE: This field identifies the gender identity of the person. GTVGNDR
    SPBPERS_PPRN_CODE VARCHAR2(4 CHAR) PPRN CODE: This field identifies the personal pronoun of the person.
    SPBPERS_PREF_MIDDLE_NAME VARCHAR2(60 CHAR) PREFERRED MIDDLE NAME: This field maintains the preferred middle name associated with person.
    SPBPERS_PREF_LAST_NAME VARCHAR2(60 CHAR) PREFERRED LAST NAME: This field maintains the preferred last name associated with person.
